

## 📝 AIM
- To understand and apply **basic CSS styling and animation** in a React project.  
- To explore how **selectors, transitions, keyframes, and media queries** enhance user interface design.  
- To demonstrate **hover effects and responsive animations** using CSS.

---

## 📖 THEORY
- **CSS Styling**: Cascading Style Sheets define the presentation of HTML elements, controlling layout, colors, spacing, and responsiveness.  
- **Selectors**: Classes like `.logo` or `.card` target specific elements for styling.  
- **Transitions & Filters**: Smooth visual changes (e.g., drop shadows) are achieved using `transition` and `filter`.  
- **Animations**: `@keyframes` defines a sequence of transformations (e.g., rotation).  
- **Media Queries**: Allow conditional styling based on user preferences (e.g., reduced motion accessibility).  
- **Responsive Design**: `max-width`, `margin: auto`, and `padding` ensure content adapts to different screen sizes.

---

## ⚙️ PROCEDURE
1. **Setup Project**: Initialize a React project (e.g., with Vite).  
2. **Define Root Styling**:  
   - Limit width (`max-width: 1280px`).  
   - Center content (`margin: 0 auto; text-align: center`).  
   - Add padding for spacing.  
3. **Style Logo**:  
   - Set height and padding.  
   - Add hover effects with `filter: drop-shadow`.  
   - Differentiate React logo with a distinct color shadow.  
4. **Add Animation**:  
   - Create `@keyframes logo-spin` for rotation.  
   - Apply infinite spin animation to the second logo link using `a:nth-of-type(2)`.  
5. **Accessibility Consideration**:  
   - Use `prefers-reduced-motion` media query to respect user system preferences.  
6. **Style Cards & Text**:  
   - `.card` adds padding for content blocks.  
   - `.read-the-docs` styles documentation links with muted color.

---

## 🎯 LEARNING OUTCOMES
- Ability to **apply CSS properties** for layout, spacing, and alignment.  
- Understanding of **hover effects and transitions** for interactive UI.  
- Knowledge of **CSS animations** using keyframes.  
- Awareness of **responsive and accessible design practices** (media queries).  
- Practical skills in **integrating CSS with React components** for modern web development.
